About Me



I am William A. Feliciano Rios, a Computer Science graduate from Southern New Hampshire University.I'm a passionate full stack developer with hands-on experience building web applications using a variety of modern stacks including .NET with React, Spring Boot with React & TypeScript, and the MERN stack. Throughout my studies, I established a strong foundation in programming with languages such as Java, JavaScript Python, and C++. I also gained expertise in full-stack development, which has equipped me with a well-rounded skill set. Over the duration of my program and outside of it, I learned how to collaborate with others using Git and GitHub. For example, last December, I was able to do a freelance project with another developer from Australia while I was in Buffalo, New York. With collaboration between me and the other developer, we delivered the project on time, with positive remarks from the client. These experiences have improved my teamwork and communication skills with technical and non-technical people.


During my program, I explored data structures and algorithms, and in one of my enhancements, I used these skills and implemented the logic for a sorting algorithm to work with my dynamic React table of inventory items. I have also worked on projects utilizing relational databases, like SQLite and NoSQL databases, such as MongoDB, giving me a solid understanding of indexing, normalization, and CRUD operations. On the security side, I created an authentication system using JSON Web Tokens (JWT), complete with middleware for user verification and access control. With these skills, I am able to demonstrate my Full-Stack Developer capabilities and my ability to design and build scalable multi-platform systems.


My Portfolio reflects these skills and illustrates my readiness for a career in computer science. The projects in my Portfolio all share a common goal: transforming basic applications into professional-grade full-stack solutions. The inspiration for my portfolio came from an inventory management application I developed in my mobile architecture and programming course. This was an Android mobile app that used a per-installation SQLite local database, and I re-architected it to use instead a centralized web API built with NodeJS and Express and secured with JSON Web Tokens. So that users may share the same data even when using two different phones. To improve the limitation of the Android platform, I built a new React and Typescript web client, which enables the app to be used on IOS and Web platforms. This frontend employs modern practices such as reusable components, separation of concerns, and state management with libraries like Tan Stack Query. Moreover, I refactored the Android application to use a library called “OkHTTP” to make HTTP requests to the centralized API and use the data across devices. I also kept a local per-install database to store users' phone numbers to enable the capability of low-stock SMS notifications while maintaining user privacy.


These projects demonstrate my capability to design and implement solutions across the entire software stack—frontend, backend, and database—while prioritizing performance, usability, and security. My portfolio showcases my growth as a problem solver.
